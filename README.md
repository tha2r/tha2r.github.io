# Thaer M. Alhanafi – Portfolio Website

Welcome to the source code for my professional portfolio website! This site serves as a comprehensive showcase of my 15+ years of experience in Software Engineering, Database Administration, System Engineering, and Penetration Testing.

## ✨ Features

- **Modern Architecture**: Fully custom HTML, CSS, and Vanilla JavaScript with zero heavy front-end frameworks.
- **Dynamic Theming**: Built-in Light and Dark modes featuring automatic `localStorage` state persistence and seamless transitions.
- **Interactive Navigation**: Minimalist dot-based "presentation slide" scrolling with smooth mandatory scroll-snapping, floating Up/Down arrows, and live active-state tracking.
- **Keyboard Friendly**: Custom arrow-key event binding (Up/Down) to enhance user accessibility during section-by-section navigation.
- **Contact Form**: Integrated with Formspree for a serverless, seamless email form submission flow via AJAX.

## 🎨 Design

The user interface focuses on a clean, dynamic, and glassmorphic aesthetic:
- **CSS Architecture**: Exclusively relies on CSS root variables for instant theme toggling and maintainability.
- **Animations**: Smooth micro-interactions and reveal-on-scroll functionality using the native Javascript `IntersectionObserver` API.
- **Responsiveness**: Fully responsive CSS Grid and Flexbox design scaling beautifully across all mobile, tablet, and ultra-wide displays.

## 🚀 Installation & Usage

Because this is a static site built with vanilla web technologies, there is no build step or package manager required!

1. Clone this repository: `git clone https://github.com/tha2r/tha2r.github.io.git`
2. Simply open `index.html` in any modern web browser.
